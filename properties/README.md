# Properties

### NAME
The name of a feature, typically in English.

### NAME_*
The localized name of a feature, where the * in the property name is replaced by the translated language's two-character ISO language code.

### TYPE
The designation for a given feature. Example values are 'Nature Reserve', 'National Park', and 'Protected Area'.

### TYPE_*
A translated designation for a given feature, where the * in the property name is replaced by the translated language's two-character ISO language code.

### ISO_ADMIN0
The two-character ISO country code for the feature's parent country.

### ISO_ADMIN1
The two-character ISO country code for the feature's parent first-level administrative subdivision.

### ID_*
A concordance identifier, where the * in the property name is replaced by the source name. For example, the 'ID_CDDA' property contains a concordance value for the CDDA database.

### LINK_*
A URL link, where the * in the property name is replaced by the source name. For example, the 'LINK_WK' property contains a link to the feature's Wikipedia page.

### LAT
The inner-latitude value of the feature's geometry, typically sourced from Mapshaper.

### LON
The inner-longitude value of the feature's geometry, typically sourced from Mapshaper.
